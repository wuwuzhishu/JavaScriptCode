# 课程说明

>此内容为软件技术专业的课程《JavaScript程序设计》的内容

## 学习目的

本课程主要是在HTML+CSS的基础上利用JavaScript给网页添加动态的特效。

## 工作岗位

面向的工作岗位是Web前端工程师。

## 学习计划

1. 课程第一部分首先学习BootStrap框架，通过此框架理解如何来构建响应式网站，理解栅格系统以及媒体查询，在此基础上学会BootStrap的全局CSS样式以及组件，关于这一部分以一个仿优达学城网站项目来贯通知识点。
2. 课程第二部分重点学习JavaScript的语法知识（这一部分有了Java的基础之后相对来说比较容易接受，大部分语法都是一样的，重点关注差异的地方），重点学习DOM、BOM以及函数式编程等JavaScript的特性。
3. 课程第三部分学习JQuery，然后利用JavaScript和JQuery第一部分的仿优达学城网站加点特效，完善仿优达学城网站。
4. 课程第四部分学习前后端交互的知识，学会从服务器端读取数据在前端显示。

## 考核要求

考核方式采用过程考核，期末不进行考试，需要每个人提交两个作品。
